Folder Purpose
api/ All HTTP requests (Axios instance, API methods for blogs, users, etc.).
components/ Reusable UI elements organized by feature or purpose.
layouts/ Shared page layouts (e.g., Navbar + Footer wrapper).
pages/ Actual route-level components (Home, BlogDetail, etc.).
context/ Handles global states like authentication.
hooks/ Custom hooks for reusable logic (auth, API fetching, etc.).
redux/ For global state management using Redux Toolkit or Zustand.
services/ Utility services (auth, token, or local storage).
utils/ Helper functions (slug generator, date format, constants).
styles/ Tailwind setup and optional global CSS themes.
